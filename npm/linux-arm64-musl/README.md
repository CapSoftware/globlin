# @globlin/linux-arm64-musl

This is the **Linux ARM64 (musl/Alpine)** binary for [globlin](https://www.npmjs.com/package/globlin).

This package is automatically installed when you install `globlin` on a Linux ARM64 system with musl (e.g., Alpine Linux ARM64).

**Do not install this package directly** - install `globlin` instead.

```bash
npm install globlin
```
