# @globlin/linux-x64-gnu

This is the **Linux x64 (glibc)** binary for [globlin](https://www.npmjs.com/package/globlin).

This package is automatically installed when you install `globlin` on a Linux x64 system with glibc.

**Do not install this package directly** - install `globlin` instead.

```bash
npm install globlin
```
