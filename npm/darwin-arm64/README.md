# @globlin/darwin-arm64

This is the **macOS ARM64 (Apple Silicon)** binary for [globlin](https://www.npmjs.com/package/globlin).

This package is automatically installed when you install `globlin` on a macOS ARM64 (M1/M2/M3) system.

**Do not install this package directly** - install `globlin` instead.

```bash
npm install globlin
```
