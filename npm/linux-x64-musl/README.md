# @globlin/linux-x64-musl

This is the **Linux x64 (musl/Alpine)** binary for [globlin](https://www.npmjs.com/package/globlin).

This package is automatically installed when you install `globlin` on a Linux x64 system with musl (e.g., Alpine Linux).

**Do not install this package directly** - install `globlin` instead.

```bash
npm install globlin
```
