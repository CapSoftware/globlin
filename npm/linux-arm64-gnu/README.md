# @globlin/linux-arm64-gnu

This is the **Linux ARM64 (glibc)** binary for [globlin](https://www.npmjs.com/package/globlin).

This package is automatically installed when you install `globlin` on a Linux ARM64 system with glibc (e.g., AWS Graviton, Raspberry Pi OS 64-bit).

**Do not install this package directly** - install `globlin` instead.

```bash
npm install globlin
```
