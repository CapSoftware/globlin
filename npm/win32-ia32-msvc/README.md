# @globlin/win32-ia32-msvc

This is the **Windows x86 (32-bit)** binary for [globlin](https://www.npmjs.com/package/globlin).

This package is automatically installed when you install `globlin` on a Windows x86 32-bit system.

**Do not install this package directly** - install `globlin` instead.

```bash
npm install globlin
```
